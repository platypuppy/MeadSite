Common printf formatting codes 

%c - characters
%s - strings (NUL-terminated C strings)
%d, %i - integers
%f - floating point (float and double)
%g - floating point (minimum digits, float and double)
%e - scientific notation (float and double)
%p - pointers (displays in hex)
%x - hexadecimal integers
%o - octal integers
%u - unsigned integers
%ld, %li - long integers
%lu - unsigned long integers
%hd, %hi - short integers
%hu - unsigned short integers

Reading (scanf) double: %lf
Reading/writing long double: %Lf
