Various representations of numbers

Only the values from 32-127 have valid character representations. Attempting to display a character representation for values less than 32 or greater than 127 will produce undefined results. This means that each computer, operating system, program, etc. may display something different. In fact, you may find that different web browsers will display this chart differently.
