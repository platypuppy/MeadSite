Here are some popular Questions and Answers for the List assignment.

  1. Question:
    Doxygen is complaing that the templated friend function operator<<() is not documented. How do I fix that?

    Answer:
    If Doxygen complains about that, it may give a warning similar to this: 

    warning: Member operator(std::ostream &os, const List &list) (friend) of class CS170::List is not documented.
    The way we will solve this is to put a copy of the Doxygen function header comment above the declaration in List.h. In the header file, you'll have something like this: 
    /*!**********************************************************************
  Outputs all of the data in a list to an output stream.

  \param os
    The stream to output the list to.

  \param list
    The list to output to the stream.

  \return
    The ouput stream (ref) that was passed in (for chaining)
************************************************************************/  
friend std::ostream& operator<< <T>(std::ostream & os, const List &list);
