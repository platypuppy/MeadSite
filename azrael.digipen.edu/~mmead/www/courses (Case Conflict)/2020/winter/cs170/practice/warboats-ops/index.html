Practice Assignment

(warboats-ops)

Information

This practice assignment is a modification of the WarBoatsOO (Object-oriented WarBoats) assignment. This practice adds several public operations to the original Ocean class. You only need to modify Ocean.h and Ocean.cpp.

  1. Copy constructor - This will be used to make copies of existing Oceans. The default copy constructor provided by the compiler is insufficient and will result in crashes.

  2. Copy assignment operator - This will allow for proper assignment. The default copy assignment operator provided by the compiler is insufficient and will result in crashes and memory leaks.

  3. Equality operator - This will allow you to compare two Oceans for equality. For purposes of this practice, two Oceans are identical if they have the same dimensions, the same number of boats placed, and all of the entries in the grid array are the same. Nothing else is considered when comparing them.

  4. Insertion (output) operator - This will allow you to print the Ocean using the << operator instead of the DumpOcean function. This is a more natural way of producing output in C++.

    You should use the Dump function from the assignment's driver as a starting point for this function. You'll need to modify it as appropriate. There will be no "customizing" the output like in the previous assignment. Make sure your output matches the output that is posted.

  5. GetNumBoats - This is a public method that the driver will call to get the number of boats that have been placed in the Ocean. This helps the driver to determine when all of the boats have been sunk.
See the driver for details of how these are used.
