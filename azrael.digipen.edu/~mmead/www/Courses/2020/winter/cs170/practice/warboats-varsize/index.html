Practice Assignment

(warboats-varsize)

Information

  1. This practice assignment is a modification of the WarBoatsOO (Object-oriented WarBoats) assignment. In the original WarBoats program, all of the boats were the same length (4). In this version, the boats can be of different sizes. The shortest boat will be of length 1, and the longest can be of any length. Of course, the longest boat will be limited to the dimensions of the ocean.

    In order to support boats of different sizes, the Boat structure needs to be modified by adding a length member to the struct: 

    struct Boat
{
  int hits;                 // Hits taken so far
  int ID;                   // Unique ID
  int length;               // Length of the boat (This is new)
  Orientation orientation;  // Horizontal/Vertical
  Point position;           // x-y coordinate (left-top)
};
    This also means that you no longer need this in Ocean.h 
    const int BOAT_LENGTH = 4;  //!< Length of a boat
    The length of the boat will be set by the driver when it calls PlaceBoat. When you add the boat to the dynamically allocated array of Boats in your Ocean class, the length will be included.

    Then, when the driver calls TakeShot, you will have to check the length of the boat being hit to determine if it is sunk. (In the original version, when the boat had 4 hits, it was sunk because all boats were of length 4.)
