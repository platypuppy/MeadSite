Inserting into a vector:

  1. Assume this call: Insert(7, 2); // Insert the value 7 at position 2

  2. You must make sure that there is enough room in the array for one additional element. If the array is full, you must grow it first (like the above diagrams).

  3. The insertion point is index 2: (size is 5, capacity is 8)

  4. For each element to the right of the insertion point, shift (copy) it over one slot to the right (working right to left).
  5. Then copy the item at the index over to the right.
  6. Finally, insert the value at the proper index: (size is now 6, capacity is still 8)
