    • Clang compiler

      To install Clang version 8.x, you need to run these commands: 

      sudo apt-get install clang
sudo apt-add-repository "deb http://llvm.org/apt/bionic/ llvm-toolchain-bionic-8 main"
sudo apt-get update
sudo apt-get install clang-8
sudo update-alternatives --install /usr/bin/clang clang /usr/bin/clang-6.0 60 --slave /usr/bin/clang++ clang++ /usr/bin/clang++-6.0
sudo update-alternatives --install /usr/bin/clang clang /usr/bin/clang-8 80 --slave /usr/bin/clang++ clang++ /usr/bin/clang++-8
      The repository is for Mint 19.x distros. Mint 19.x is based on Ubuntu Bionic, that's what the bionic means. If you are using some other version, you may have to change that to something else. See this to determine what to put there. Also, if you want a different version of Clang, you'll need to change the "8" in the lines above to the correct value. See this for examples.

      To switch between versions, run this command: 

      sudo update-alternatives --config clang

  • Doxygen - is a free, cross-platform, automated documentation generator. The version that comes with Linux Mint 19.2 is 1.8.13, and that's the version we'll be using. Simply install it like this: 
    sudo apt-get install doxygen
    Here are the commands that you will need to run on your homework (and put in your makefile) 
    ( cat Doxyfile ; echo "EXTRACT_ALL=YES" ) | doxygen -
( cat Doxyfile ; echo "EXTRACT_ALL=NO" ) | doxygen -
    You'll need to modify this configuration file for each assignment. You'll also need to remove the file extension.

    You will also need to create a PDF file from the Doxygen-generate files. This will require a few other programs to be installed, which is trivial to do under Linux. You should install these: 

    sudo apt-get install doxygen-latex evince
    If any other dependencies are required, simply install those as well. If you want the full experience, install the full version: 
    sudo apt-get install texlive-full
    Realize that this could install over 3 GBs of files. There's also a nice editor/IDE for creating documents called Texmaker that works on Linux, Windows, and Mac. I use that to create all of the quizzes and exams. Latex is one of the industry-standard tools for creating high-quality technical documents, so learning the basics will give you skills outside of just creating documentation for your code in CS180.

    Here is a sample1.tex file to check your full installation. (Remove the .txt extension to use the file.) The sample has chemistry diagrams and musical notation, so you'll have to at least install these other packages (if you didn't do a full install): 

    sudo apt-get install texlive-science texlive-music abcm2ps ps2eps

    Then run this command to produce a PDF: 

    pdflatex -shell-escape sample1.tex
