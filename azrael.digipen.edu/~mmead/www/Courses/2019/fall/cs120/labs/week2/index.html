  • Command line for compiling this lab assignment: 
    gcc -Wall -Wextra -Werror -ansi -pedantic marathon.c -o marathon
  • Running your program and capturing the output to a file: 
    marathon > output.txt
  • Comparing your output with the expected output: 
    diff output-master.txt output.txt --strip-trailing-cr
    The --strip-trailing-cr option above is usually only needed if you are using Windows. Mac and Linux systems shouldn't need it. (It doesn't hurt, if you use it, though.)
