  • You are to create a program that prompts the users for 2 dates and then prints out if the first date comes before or after the second date. Here are some interactive examples when executing the program. (The numbers in RED were typed in by the user and then the Enter key was pressed.) 
    Enter the first date: (mm/dd/yyyy) 07/25/2018
Enter the second date: (mm/dd/yyyy) 12/01/2018
07/25/2018 comes before 12/01/2018.

    Enter the first date: (mm/dd/yyyy) 12/01/2018
Enter the second date: (mm/dd/yyyy) 04/23/2020
12/01/2018 comes before 04/23/2020.

    Enter the first date: (mm/dd/yyyy) 04/24/2017   
Enter the second date: (mm/dd/yyyy) 04/22/2017
04/24/2017 comes after 04/22/2017.

    Enter the first date: (mm/dd/yyyy) 09/21/2023
Enter the second date: (mm/dd/yyyy) 09/21/2023
09/21/2023 and 09/21/2023 are the same date.

    

There are only 3 possible outcomes:

    

      
	
    1. The first date comes before the second date.
      2. 
	
    2. The first date comes after the second date.
      3. 
	
    3. The first date is the same as the second date.
      4. 

    

    

You will create a file called dates.c and implement it there. It's really not 
a difficult task. You will have to use some conditional (if) statements, of course. 
Pay attention to the formatting. You will need to use the diff tool and redirection 
that you learned earlier. Don't forget that you may need to use 
the --strip-trailing-cr switch with diff.

    

    
You don't have to worry about invalid input at this point. (That requires a lot
more code and logic to deal with.) You can assume that the user will only be
entering valid dates. That is, 3 integers (base 10), formatted exactly as shown.
There will be 2 digits for the month and day, and 4 digits for the year, all
separated by a forward slash with no extra spaces. Each date will contain exactly
10 characters.


    

    

You'll do something like this for each date:

      /* Prompt the user to enter the first date */
printf("Enter the first date: (mm/dd/yyyy) ");
scanf("%d/%d/%d", [variables for the first date here]);
