You will be creating two functions during this lab: print_cubes and print_pythagorean_triples
  1. Write a function called print_cubes that will print all 3-digit integers that have the cube property. The prototype looks like this: 
    void print_cubes(void);
    An integer has this property if the cube of its digits add up to the number itself. For example, 153 has this property: 
    153 = 13 + 53 + 33 = 1 + 125 + 27 = 153
    The output of the program should look like this: 
    153 has the cube property. (1 + 125 + 27)
370 has the cube property. (27 + 343 + 0)

etc...
    Hint: To extract the digits of the number, use the division and/or the modulus operators: / %

    Pseudo-codeMore detailed pseudocode
    
/* For each 3-digit number                 */
/*   Extract the three digits              */
/*   If the sum of the cubes of the digits */
/*     equals the number, print the number */

    
/* For each 3-digit number                      */
/*   Extract and save the the first digit       */
/*   Extract and save the second digit          */
/*   Extract and save the third digit           */
/*   Cube first digit                           */
/*   Cube second digit                          */
/*   Cube third digit                           */
/*   Add up the cubes                           */
/*   If the sum equals the number then          */
/*     Print the number                         */

    Remember that integer arithmetic will always truncate the result if it is not an integer.

    To test all 3-digit numbers, you will need to loop from 100 up to and including 999. (That's all of the 3-digit numbers.)

    The easiest way to extract the last digit of a number is to use the modulo operator with the value 10. For example, 123 % 10 is 3, which is the last digit in the number. This technique will always give you the last digit of any integer.

    To extract the first digit, just divide the number by 100, so 123 / 100 is 1.

    Extracting the middle digit is only slightly more involved using both operators, but you should be able to figure out a way to do that.

  2. Write a function that finds Pythagorean Triples. The prototype looks like this: 
    void print_pythagorean_triples(int low, int high);
    The inputs are the range [low, high) of values that you should try. For example, this call: 
    print_pythagorean_triples(1, 100);
    would try all of the integers from 1 up to 99. The range includes the low number, but not the high number, hence the range: [low, high)

    Three integers are Pythagorean triples if they have the property that the square of the largest value is equal to the sum of the squares of the two smaller values. For example, the integers (3, 4, and 5) are triples as well as others: 

    32 +   42 =  52  (  9 +   16 =   25)
52 +  122 = 132  ( 25 +  144 =  169)
282 + 452 = 532  (784 + 2025 = 2809)
    There are an infinite number of Pythagorean triples and many ways to calculate them. The purpose of this assignment is to practice your coding with for loops, so that's how you should do it. (Hint: you will use 3 nested for-loops. Use i, j, and k for the loop counters.) The output of the program should look like this:

int main(void)
