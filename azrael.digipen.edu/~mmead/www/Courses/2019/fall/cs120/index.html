    Required Software (free)

    All of the software that is required for this course is installed on all of the computers at Digipen. However, for those of you that wish to also work on your own computers (recommended), these are the things you will need to install and configure on your own computer.

    • Windows 64-bit C compiler (MinGW) This is the compiler that we will be using for all of the work in CS120. There are detailed instructions on downloading, installing, and configuring the software here. Note that this will also install a C++ compiler, which you will need next semester in CS170.

    • Cygwin Utilities - Download the Cygwin64 programs and put them on your computer. There's a document readme-2017-cygwin-install.pdf that explains what to do.

      The zip file contains many command-line utilities that you will use throughout your time at Digipen. The zip file is about 530 MB and extracts to about 1.5 GB. Creates a folder named cygwin64 on the C: drive.

      • Don't forget to add C:\cygwin64\bin directory to your PATH. If you're new to computers or Windows, there's a nicer GUI (Graphical User Interface) program called Windows Environment Variables Editor that makes it easier to manipulate the PATH.
      • Understanding the PATH variable from wikipedia.
      • Editing the PATH variable on Windows 7, 8, and 10.
      • Once you've got Cygwin installed in Windows and in your PATH, you can type this to see a better view of your PATH from the command line: 
        path | tr ; "\n"
        This will list all of the directories in the PATH with each on a separate line. That is:
        TokenMeaning
        pathDisplays the PATH as one long line
        |A vertical bar (pipe symbol), usually on the key with the backslash (\)
        trThe translate command
        ;A semi-colon (usually to the right of the letter 'L' on the keyboard). On Linux and Mac OS X, use a colon : instead.
        "\n"A double quote, backslash, lowercase letter 'N', double quote
    • C compiler for non-Windows users:

      Version 8.1.0 is what is installed with MinGW on the computers at Digipen. For non-Windows users, if you can't get version 8.1.0, any 8.x.x version will probably be fine. We won't be using any of the advanced features of C that are in 8.x, but you do want to have the closest version possible so that you will get the same errors and warnings that 8.1 emits.

      For Mac people, you may have to install it (using homebrew) as the default compiler for C and C++ on Mac is Clang. If you are new to software development, you may need to seek help from other Mac users. (I'm not a Mac user, although I pretend sometimes!)

      For Linux people, I recommend Linux Mint. The latest version comes with version 7.x, but it can easily be upgraded. Here's some information on installing multiple versions. However, any distribution that allows you to use version 8.x of the compiler should work.

      Note that these installations will also install a C++ compiler, which you will need next semester in CS170.

    Notes

    • Fundamentals
      • Creating a simple program Editing, compiling, linking, executing
    • Formatted Input/Output Introduction to printf, scanf, format specifiers
    • Expressions and Operators Precedence, associativity, order of operations, assignments, side-effects
    • Conditionals Booleans, relational/logical operators, if, else, switch, case
    • Iteration Looping and repetitions, while, do, for
    • Functions Prototypes, parameters, arguments, call-by-value, scope, lifetime
      • Larger programs multiple file programs, separate compilation/linking
    • Topic: Random Numbers Pseudo-random number generators (PRNG), seeding (Pages 686-687 in 2nd edition of the textbook, pages 570-571 of first edition)
    • Arrays one-dimensional, sizeof operator, subscripting
    • Data Types integral types, signed/unsigned integers, long/short integers, characters, floating point types, long doubles
    • Pointers address operator, indirection/dereference operator, arrays vs. pointers
    • Strings string literals, variables, NULL character, character arrays vs. strings, string functions
    • File I/O FILE, opening/closing, reading/writing text files, redirection
    • Structures struct, pointers to structures, structure operators
    • Dynamic Memory Allocation malloc/free
    • Introduction to linked-lists A dynamic and flexible data structure.
    • Complex declarations Pointers, arrays, functions and function pointers.
    • More Arrays Two-dimensional arrays, pointers and 2D arrays.
    • Visibility and Lifetime Linkage (internal vs. external), lifetime (storage classes)
    • The topics below will NOT be on the final exam. Only the topics listed above will be eligible for the exam.
      • Variadic Functions like printf and scanf
      • More Structures alignment, unions, bit-fields.
        • Record-based File I/O using structured data files.
      • Runtime Environment Command line arguments, environment variables, exec functions
      • RunTime Library Lots of functions for you to learn.
