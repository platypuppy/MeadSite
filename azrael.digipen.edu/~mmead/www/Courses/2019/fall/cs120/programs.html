Chapter 13

  1. What does the following program print? 
    #include <stdio.h>

int main(void)
{
  char s[] = "Hsjodi", *p;

  for (p = s; *p; p++)
    --*p;

  puts(s);

  return 0;
}
  2. Let f be the following function: 
    int f(char *s, char *t)
{
  char *p1, *p2;

  for (p1 = s; *p1; p1++)
  {
    for (p2 = t; *p2; p2++)
      if (*p1 == *p2)
        break;

    if (*p2 == '\0')
      break;
  }
  return p1 - s;
}
    1. What is the value of f("abcd", "babc")?
    2. What is the value of f("abcd", "bcd")?
    3. In general, what value does f return when passed two strings s and t?
